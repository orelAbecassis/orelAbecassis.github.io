Présentation du projet

L'objectif de ce projet était de créer trois jeux à savoir le mémory, le Jackpot et le rapido. Nous avions un document éxpliquant le travail attendu avec des exemples.

Nos missions

Pour le jackopt qui été un jeux a développer seul. j'ai choisi le thème du théatre, pour cela j'ai eu besoin de créer : Utilisation du scss pour le menu du jeux, un rideau d'entrée, une image intégrée dans le titre, un son d'ambiance, un éclairage lors du survol du bouton "Lancer", J'ai alors commencer à mettre en place la partie fonctionel de la machine à sous, puis, dans un second temps, j'ai travailler le desing (images, fond, mise en forme...)

Nos missions

Pour le memory nous avons suivi notre charte graphique, grâce au logiciel krita nous avons créer quelques cartes dans le théme des emoji. Dans cette optique nous avons gardé un théme sombre pour la page, tout comme le serait un mode sombre d'un jeu sur mobile. Pour ce jeux nous avions comme idées : Avoir des cartes emoji, Animer le retournement des cartes, Animer la selection de cartes, Animation sonore, Mettre la selection du niveau et le jeu sur deux parties différentes de l'écran, Un mode : limite de temps, Un mode : limite de coups.

Les outils

Pour ce projet, nous étions par groupe de deux, nous nous sommes départagés les tâches. Nous avons travaillé avec les langages suivant HTML, CSS et JS.

Conclusion

Pour ce projet, nous devions créer un jeu sur trois ce qui a été le premier projet sur lequelle ou nous devions apprendre seul. Notre professeur de développement Web nous a fourni un cahier des charges et ce fus à nous de concevoir le format et le thème du projet. Nous avons dû apprendre de nouvelles mèthodes et de nouvelles bibliothèques Java Script pour réaliser le projet. Pour les deux autres jeux, nous les avons réalisés en équipe et cela nous a permi de cohopérer, à communiquer ainsi de gérer la répartition des tâches pour équilibrer les charges de travail.
