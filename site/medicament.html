Présentation du projet

Lors de cette 2 eme année, nous avons appris à utiliser le Modele Vue Controleur pour les projets en JAVA avec notre professeur de développement d'Application, à travers diverse TP.

Notre professeur nous a demandé de créer une application pour différentes gestions et en équipe. Nous avons choisi la gestion de médicament qui est composé de plusieurs fonctionalités repartit en plusieurs categories : les médecins qui doivent se connecter, les différents médicaments avec leurs interactions ainsi que les prescriptions pour chaques individu.

Pour la gestion de médicaments et des individus les fonctions demandés sont : lister, modifier, ajouter. Pour les interactions de tout les médicaments la fonction était de lister tout les médicaments, d'en sélectionner un pour savoir s'il est dangereux de le prendre en même temps qu'un autre ou pas. Un médecin a la possibilté de préscrire une ordonnance.

Comme nous avons établi ce projet en équipe, pour se partager les différentes tâches, nous avons créer un gestionnaire de projet avec le Logiciel Trello.

Lorsqu'on entre sur l'onglet des médicament, on trouvera tout la liste des médicaments déjà référencés. On pourra les modifier simplement en séléctionnant un et cliquer sur le bouton " Modifier" qui vous conduira sur une autre page avec une formulaire avec les informations de ce médicament.

La liste des médicaments

L'intéraction des médicaments

Un médecin peut préscrire des ordonnances, mais avant il faut être sur de ne pas donner des médicaments dangereux s'ils sont pris en interval, pour vérifier il suffira de sélectionner un médicament. Nous avons créer trois tableaux : le premier sera la liste de tout les médicaments, le deuxiéme affichera un ou plusieurs médicaments dangereux à prendre avec celui-ci et le troisième affichera au contraire ceux qui qui peuvent être pris avec celui-ci.

La page d'interaction

Les types d'individus

Une fois que le médecin a vérifié les intéractions, il doit vérifier si le type de personne (bébé, enfant, femme ou homme) à qui il va prescrire existe bien dans le fichier, parce que les quantités ou poids des médicaments ne sont pas les mêmes pour tout le monde. Pour cela, nous avons créer une page avec un tableau qui permettra de les lister, avec un bouton ajouter ou modifier.
