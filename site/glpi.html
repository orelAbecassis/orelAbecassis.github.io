GLPI

Projet de Deuxiéme année : FEVRIER 2022

Présentation du projet

Lors de ma deuxieme année de BTS, on nous a appris l'utiliter de GLPI. C'est un Gestionnaire Libre de Parc Informatique, c'est un logiciel en libre service.

C'est un gestionnaire qui va nous permettre de réferencer tout nos différentes machines physique présntent dans l'entreprise (ordinateurs, périphériques, serveurs, licences, messagerie ...). Ce gestionnaire permettra d'avoir une meuilleure visibilité. GLPI permet d'effectuer des tickets d'indicents.

Installation

Notre professeur nous a expliqué la théorie de son utilisation et nous a montré la démo qui est proposé sur le site officiel de GLPI, mais j'ai décidé d'en savoir plus en l'installant sur une Machine Virtuel avec VirtualBox sous Debian puis avec une autre VM sous Windows j'ai crée le client.

Etapes

GLPI est un logiciel codé en PHP, pour pouvoir de lire, j'ai tout d'abords mis en place un server Web "Apache2". GLPI fonctionne avec une base de donnée, j'ai donc installé un serveur de base de donnée " Mysql", avec PHP la version 7.4 et pour finir,j'ai installé la dernier version de GLPI.

Conclusion

J'ai décidé d'installer GLPI pour en savoir un peu plus et comprendre son fonctionnement. Ce TP de réseau m'a appris comment utliser Linux, les commandes à utiliser et comment intéragire avec ma machine physique pour effectuer des captures d'écrans mais avec une autre VM qui est le client celle qui utilise GLPI.

Je suis très satisfaite du résultat, j'ai pu effectuer un peu de réseau, j'ai rencontré quelques problémes comme pour la création des VM, trouver les bonnes commandes et surtout je les ai souvent éxécuter sur les mauvais chemins. Pour pus de facilité, j'ai édité une documentation pour les prochaines fois ou si cela pourrait aider autruit.
